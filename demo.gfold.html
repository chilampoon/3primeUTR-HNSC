GFOLD: a generalized fold change for ranking differentially expressed genes from RNA-seq data

A technique for estimating fold change that takes into account the uncertainty of gene expression measurement by RNA-seq. The read counts are modeled by Possion distribution. GFOLD not only measures the fold change but also captures the small variance of the posterior distribution of log2 fold change for the highly expressed genes

Follow the steps on GFOLD bitbucket

countToFpkm <- function(counts, effLen) {
  N <- sum(counts)
  exp(log(counts) + log(1e9) - log(effLen) - log(N) )
}

read_cnt1$RPKM <- mapply(countToFpkm, read_cnt1[['Read Count']], read_cnt1[['Gene exon length']])

Analyze the result

The GFOLD value could be considered as a reliable log2 fold change. It is positive/negative if the gene is up/down regulated. The main usefulness of GFOLD is to provide a biological meanlingful ranking of the genes. The GFOLD value is zero if the gene doesnt show differential expression.

Distribution of the measures

library(ggplot2)

# GFOLD
ggplot(gfold, aes(x=GFOLD)) + 
  geom_density() +
  theme_classic()

# logFC
ggplot(gfold, aes(x=log2fdc)) + 
  geom_density() +
  theme_classic()

## manual MA plot
ma.df1 <- data.frame(log2fc = gfold$log2fdc, meanCnt = log2(sqrt(gfold[,'1stRPKM'] * gfold[,'2ndRPKM'])))
ma.df2 <- data.frame(log2fc = gfold$log2fdc, meanCnt = 0.5 * (log2(gfold[,'1stRPKM'] + 1) + log2(gfold[,'2ndRPKM'] + 1)))
ma.df1 <- ma.df1[ma.df1$meanCnt != -Inf,]

ggplot(ma.df1, aes(x=meanCnt, y=log2fc)) +
  geom_point(alpha=0.3, size=1.8) +
  geom_hline(aes(yintercept=0), size=0.6) +
  labs(title="log2 RPKM") +
  theme_classic()

ggplot(ma.df2, aes(x=meanCnt, y=log2fc)) +
  geom_point(alpha=0.3, size=1.8) +
  geom_hline(aes(yintercept=0), size=0.6) +
  labs(title="(log2 + 1) RPKM") +
  theme_classic()
